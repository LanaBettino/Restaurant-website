# Restaurant Website

## Overview

This is a responsive restaurant website created using HTML, CSS, and JavaScript. The website allows customers to view menus, search for meals, and navigate through different food categories.

## Features

* Home page with slideshow background
* Navigation menu
* Search bar
* Menu categories (Burgers, Pizza, Drinks, Desserts)
* Responsive design for mobile and desktop
* Contact information section

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Installation

1. Download or clone the project files.
2. Open the project folder.
3. Open `index.html` in a web browser.

## Project Structure

* `index.html` - Home page
* `about.html` - About Us page
* `menu.html` - Main menu page
* `style.css` - Website styling
* `script.js` - JavaScript functionality
* `images/` - Images used on the website
